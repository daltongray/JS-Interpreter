JS-Interpreter Object.defineProperty Demo

While Proxy is an ES6 feature, Object.defineProperty can be used to achieve a similar proxy effect per property. This can be used to simulate complex objects from within the interpreter. This demo achieves the illusion of passing a complex object into the interpreter while using append code, but any api could use a similar process to return objects with custom getters and setters in this way.

Click Parse, then click Toggle Indicator. Open your browser's console for errors.
